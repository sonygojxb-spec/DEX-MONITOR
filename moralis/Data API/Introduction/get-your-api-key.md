# Get Your API Key

> Learn how to get your Moralis API Key

To start using Moralis products, you’ll need an API key to authenticate your requests.

Follow this quick guide to create your account and retrieve your API key- then you'll be ready to make your [first request](/data-api/first-request).

## 1. Create Your Account

Sign up for a **free Moralis account** by visiting the [Moralis Dashboard](https://admin.moralis.com/register).

<img src="https://mintcdn.com/moralis/z0RKSaPe_EdRhxsG/images/moralis-login.png?fit=max&auto=format&n=z0RKSaPe_EdRhxsG&q=85&s=5a6bc8dfdb01397425ec88b49d4bdf0a" alt="Create Account Web" title="Create Account Web" style={{ width:"100%" }} width="1512" height="968" data-path="images/moralis-login.png" />

## 2. Navigate to Data APIs

Once registered, navigate to the [Data APIs section](https://admin.moralis.com/data-api) in the dashboard.

<img src="https://mintcdn.com/moralis/z0RKSaPe_EdRhxsG/images/moralis-dashboard.png?fit=max&auto=format&n=z0RKSaPe_EdRhxsG&q=85&s=c94279f9375fad567c87872171cff4ea" alt="Moralis Dashboard" width="1510" height="964" data-path="images/moralis-dashboard.png" />

## 3. Copy Your API Key

Click the **copy** icon next to your API key to add it to your clipboard.

<img src="https://mintcdn.com/moralis/z0RKSaPe_EdRhxsG/images/moralis-data-api.png?fit=max&auto=format&n=z0RKSaPe_EdRhxsG&q=85&s=3c729d301882d20c8eff1bb961e0f025" alt="Moralis Data APIs" width="1510" height="964" data-path="images/moralis-data-api.png" />

## 4. Manage API keys (optional)

You can also manage your keys from the **API Keys** section - including creating new keys or rotating existing ones.

<img src="https://mintcdn.com/moralis/z0RKSaPe_EdRhxsG/images/moralis-api-keys.png?fit=max&auto=format&n=z0RKSaPe_EdRhxsG&q=85&s=a5bc1c4c72a465c3d47eb872fa164648" alt="Moralis Api Keys Pn" width="1510" height="964" data-path="images/moralis-api-keys.png" />

That's it - you're now ready to make your [first API request](/data-api/first-request)
